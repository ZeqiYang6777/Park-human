Full Opaque Interactive Water and Waterfall - README
--------------------------------------------------------------

Overview
--------------------------------------------------------------
The Full Opaque Interactive Water and Waterfall package is a stylized water solution designed for mobile, VR, and stylized projects. It delivers high-performance, fully opaque shader effects optimized for Unity's Universal Render Pipeline (URP). This pack includes two customizable shader materials (water and waterfall), a vertex paint system to control foam placement, and a target tracking script to add dynamic turbulence and trail effects in the water.

--------------------------------------------------------------
Features
--------------------------------------------------------------
* Vertex Paint Tool Script (for foam painting)  
* Customizable Water Material  
* Customizable Waterfall Material  
* TargetPosition Script  
* Dynamic Trail and Turbulence Effects  
* Waterfall-Induced Turbulence  
* Steam VFX  
* 100% Opaque Rendering  
* URP & Shader Graph  
  Custom Shader Graph compatible with URP  
* Lightweight & Optimized  
  Designed for high performance on all platforms  
* Demo Scene Included  

--------------------------------------------------------------
Vertex Paint Script
--------------------------------------------------------------
The Vertex Paint tool allows you to draw foam areas directly onto the water surface for more artistic control.

How to Use the Vertex Paint System
--------------------------------------------------------------
1. Import the Water prefab into your scene  
2. Go to: Tools > Vertex Paint  
3. Use **White** to paint foam, **Black** to remove it  
4. Adjust **Brush Size** to control the paint area  
5. Adjust **Strength** to change the painting intensity  

--------------------------------------------------------------
TargetPosition Script
--------------------------------------------------------------
The TargetPosition script allows the shader to track a specific object (like your player) to create dynamic trail effects on the water.

How to Use the TargetPosition Script
--------------------------------------------------------------
1. Add the TargetPosition script to your main character  
2. In the Inspector, assign the character to the **Target** field of the script  
3. Note: Only one target can be assigned at a time  

--------------------------------------------------------------
Installation
--------------------------------------------------------------
1. Download and import the Full Opaque Interactive Water and Waterfall package  
2. Drag the Water prefab into your scene  
3. Use the Vertex Paint Tool to paint foam effects where needed  
4. Attach the TargetPosition script to your character  
5. Assign the character to the Target field  
6. Press Play and observe the dynamic water interaction
